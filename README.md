# ViPER4Android-LineageOS

ViPER4Android Reverse Engineered Fork optimized for LineageOS

## Description

ViPER4Android Reverse Engineered Fork is an audio enhancing software to provide everyone with a better audio experience.

## Features

- Audio effects processing
- Compatible with LineageOS
- System-level integration
- Easy installation via Magisk

## Installation

1. Flash the module via Magisk Manager
2. Reboot your device
3. Open ViPER4Android app to configure settings
